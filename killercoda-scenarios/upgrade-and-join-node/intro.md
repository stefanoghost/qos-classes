# Kubernetes CKA Scenario

You are working as a Kubernetes administrator in a production environment.

One of your coworkers informed you that a node is not part of the cluster and is running an outdated Kubernetes version.

Your task is to investigate and fix the issue.

## Environment

- You are already connected to the control-plane node
- A worker node named **node01** is available
- SSH access is enabled

Good luck!
