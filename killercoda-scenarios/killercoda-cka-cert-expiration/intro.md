# CKA Practice: Check Kubernetes Certificate Expiration

In this scenario you will work with Kubernetes control plane certificates, similar to a real CKA exam task.

You need to inspect the `kube-apiserver` server certificate, record its expiration date, and write the `kubeadm` command that would renew that certificate.

The cluster has been prepared with `kubeadm` on a single control-plane node.
